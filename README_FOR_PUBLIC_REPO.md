# 日语学习每日推送系统 - 公开说明仓库

这是一个自动化日语学习系统的**公开说明文档**。

---

## 📖 这个系统是什么？

一个完全自动化的日语学习工具，每天早上 8:00 推送日语学习内容到飞书群组。

**特点**：
- ✅ 90 天从零到 N5 水平
- ✅ 完全自动化，无需人工干预
- ✅ 结合二次元文化，学习更有趣
- ✅ AI 生成内容，质量高
- ✅ 每天只需 10-15 分钟

**学习内容**：
- 🔤 五十音每日复习
- 🎯 今日词汇（带罗马音）
- 📝 语法详解（二次元例句）
- 🎮 实战演练
- 💡 文化小知识

## 📂 项目结构（供参考）

如果你想自己搭建，需要准备：

```
japanese_daily/
├── syllabus.json          # 90天学习大纲
├── kana_data.json         # 五十音数据
├── daily_push.py          # 主推送脚本
├── config.py              # 配置文件
├── current_day.txt        # 学习进度记录
├── requirements.txt       # Python依赖
└── .github/workflows/
    └── daily_push.yml     # 自动化配置
```

**核心文件说明**：
- `syllabus.json`：定义每天学什么
- `daily_push.py`：调用AI生成内容并推送
- `daily_push.yml`：GitHub Actions配置（定时任务）

你可以参考这些文件自己搭建，或者联系原项目作者获取完整代码。
